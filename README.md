# 🌟 Team-Management - Fullstack Task  Web App

**Team-Management** is a feature-rich, dark-themed web app designed to help individuals and teams manage tasks, projects, calendar events, real-time chats, and more — all in one place.

---
## Live Demo Link  : https://lnkd.in/dAhBS6ie


## 🔥 Features

- 🔐 Secure authentication using Supabase
- 📋 Smart to-do lists with priorities and deadlines
- 📅 Calendar-based task organization
- 🧑‍🤝‍🧑 Create projects and assign team members
- 💬 Real-time team chat system
- 🔔 Instant notifications for invites and deadlines
- 👫 Add and manage friends/team connections
- 🧾 Resume upload and live preview in profile
- 📊 Dashboard overview of all activities
- 🌙 Fully responsive dark-themed UI

---

## 🧠 Tech Stack

- **Frontend**: React.js + Vite  
- **Backend & Auth**: Supabase  
- **Styling**: TailwindCSS  
- **Deployment**: Netlify

## 🚀 ***How to Run This Project***

> Follow the steps below to clone, install, and run the project locally.

***

## Step 1: Clone the repository

git clone <URL>
cd prject

## Step 2 : Install Dependices

npm install 

## Step 3: Setup Supabase credentials

Go to https://supabase.com and create a new project

Inside the project settings, get your SUPABASE_URL and SUPABASE_ANON_KEY

Create a .env file and add:

VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key ```

## Step 4: Create Supabase tables

Use Supabase Table Editor to create tables like:

users, todos, projects, messages, notifications, friends, etc.
Set up row-level security (RLS) and bucket for resume uploads.

## Step 5: Start the development server

npm run dev

## Step 6: Open in browser
Visit: http://localhost:5173
